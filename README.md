# Joseth Alarcón | Portafolio

Portafolio profesional desarrollado con React + Vite y Tailwind CSS para presentar proyectos, habilidades técnicas, servicios y vías de contacto de forma moderna, responsiva y lista para despliegue.

## Tecnologías usadas

- React
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Build de producción

```bash
npm run build
```

## Estructura breve

```text
src/
  components/
  data/
  App.jsx
  main.jsx
  index.css
```

## Descripción

El sitio está pensado para dos objetivos:

- Servir como portafolio profesional para postulación laboral en tecnología.
- Funcionar como vitrina para captar clientes o empresas que necesiten soluciones digitales.
